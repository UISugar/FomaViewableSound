## Requirements

* iOS 13
* Xcode 12
* Swift 5.0

## Installation

FomaViewableSound is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'FomaViewableSound'
```
